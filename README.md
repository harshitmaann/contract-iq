# ContractIQ

ContractIQ is a Financial Document Intelligence Platform built with:

- FastAPI
- PostgreSQL
- pgvector
- OpenAI Embeddings
- Retrieval-Augmented Generation (RAG)

## Current Progress

- [x] FastAPI setup
- [x] PostgreSQL + pgvector
- [x] SQLAlchemy models
- [x] Document and Chunk tables
- [ ] PDF ingestion
- [ ] Embeddings
- [ ] Vector search
- [ ] Question answering
